# Documentation Link Rot Detector

Detects and reports broken links in documentation using AI to understand context.

## Free
```yaml
- uses: walshd1/documentation-link-rot-detector@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```
